# GitMCP

[English Version](README.en.md)

## 项目简介

GitMCP 是一个基于 .NET 10 的项目，包含 GitMCP.Server 子项目。

### 子项目：GitMCP.Server

基于 C# 实现的 Model Context Protocol (MCP) 服务器，用于为 AI 助手提供 GitHub 仓库的文档和代码搜索功能。

- 自动仓库扫描
- 自动更新代码
- 动态工具注册
- MCP 协议支持
- 并发更新

### 快速开始

1. 准备仓库目录
2. 配置应用（编辑 appsettings.json）
3. 运行服务
4. 验证服务

详细说明请见 [GitMCP.Server/README.md](GitMCP.Server/README.md)
