## Introduction

GitMCP is a .NET 10 project, including the GitMCP.Server subproject.

### Subproject: GitMCP.Server

A Model Context Protocol (MCP) server implemented in C#, providing documentation and code search for GitHub repositories to AI assistants.

- Automatic repository scanning
- Automatic code update
- Dynamic tool registration
- MCP protocol support
- Concurrent updates

### Getting Started

1. Prepare repository directory
2. Configure appsettings.json
3. Run the service
4. Verify endpoints

For full details, see [GitMCP.Server/README.md](GitMCP.Server/README.md)
